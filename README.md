# MiBankito

MiBankito es una aplicación de escritorio diseñada para unificar los pagos de servicios en una sola pantalla, funcionando como una extensión integrada con el sistema MRH.

## Objetivo

Ofrecer una solución centralizada para la gestión y pago de diferentes servicios, facilitando la experiencia del usuario y optimizando los procesos administrativos, todo desde una única interfaz conectada con MRH.

## Tecnologías utilizadas

- **Lenguaje principal:** C#
- **Framework:** WPF (Windows Presentation Foundation)
- **UI:** XAML, Material Design para WPF
- **SDK:** RMH (Retail Management Hero)

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/VictorHRE/MiBankito.git
   ```
2. Asegúrate de tener instalado [.NET](https://dotnet.microsoft.com/) y Visual Studio (recomendado).
3. Instala las dependencias requeridas:
   - Agrega las librerías de RMH SDK a tu solución.
   - Agrega el paquete [MaterialDesignThemes](https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit) para WPF.
4. Abre el proyecto en Visual Studio y compílalo.
5. Ejecuta la aplicación.

## Uso

Configura las credenciales de RMH en el archivo de configuración correspondiente. Luego, ejecuta la aplicación y utiliza la interfaz para gestionar y unificar los pagos de servicios.

## Requisitos

- Windows 10/11
- .NET Framework compatible con WPF
- SDK RMH (Retail Management Hero)
- Librería de Material Design para XAML
